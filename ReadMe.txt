# پروژه پیش‌بینی قیمت خانه در کالیفرنیا

این پروژه شامل مراحل مختلفی است برای پیش‌بینی قیمت خانه در کالیفرنیا با استفاده از داده‌های خام تا مدل‌سازی و بهبود مدل.

## ساختار پروژه

- `01_data_exploration.ipynb`  
  بررسی اولیه داده‌ها، بارگذاری و تحلیل‌های مقدماتی.

- `02_preprocessing_and_split.ipynb`  
  پیش‌پردازش داده‌ها، جایگزینی مقادیر گمشده، انکودینگ، و تقسیم داده‌ها به آموزش و تست.

- `03_linear_regression.ipynb`  
  آموزش مدل رگرسیون خطی و ارزیابی آن.

- `04_model_improvement.ipynb`  
  بهبود مدل با استفاده از رگرسیون Ridge و Lasso.
